## 关键内容概览

智能驾驶功能软件平台通过对各类技术实施方案的高层次抽象，划定了六大关键功能模块：
- **传感器抽象功能**：确保来自各种传感器的数据统一处理。
- **感知融合功能**：综合处理多传感器数据，提高环境理解准确性。
- **预测功能**：分析动态对象的行为趋势，为决策做准备。
- ****决策规划功能**：核心模块，依据环境信息制定驾驶策略。
- **定位功能**：精准车辆定位，是安全驾驶的基础。
- **执行器抽象功能**：连接决策与物理世界，控制汽车响应。
